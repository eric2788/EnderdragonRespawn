# EnderdragonRespawn
終界龍定時重生插件(免費版本)

## 版本
1.13.2

## 特色
- 定時重生終界龍
- 支援離線儲存
- 重啟後可選擇直接復活或重新定時復活
- 允許的世界列表(僅限終界)
- 自定義的怪物血量條(僅限伺服器重啟後開始計算的第二只終界龍)

### [點此下載插件](http://www.mediafire.com/file/ynv720mopyrr18b/EnderDragonRespawn.jar/file)

## 權限
- er.restart - 使用 /er 的實際指令，否則顯示插件資訊

## 指令
- /er - 查看幫助
